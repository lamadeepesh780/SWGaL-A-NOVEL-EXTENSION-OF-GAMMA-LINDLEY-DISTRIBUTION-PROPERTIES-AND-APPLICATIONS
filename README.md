# SWGaL-A-NOVEL-EXTENSION-OF-GAMMA-LINDLEY-DISTRIBUTION-PROPERTIES-AND-APPLICATIONS
A new SWGaL lifetime model with full PDF/CDF, moments, MRL, and MLE-based parameter estimation. Evaluated using AIC, BIC, AICC, HQIC, KS test, and fitted plots on Wang’s and Proschan’s reliability datasets, showing strong fit and improved flexibility for survival analysis.
